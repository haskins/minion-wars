-- SUMMARY --

This simulation pits agents (referred to as minions) on teams against one another. There can be 2-4 teams competing for the win. The minions consist of Base, Healer and Attacker.


-- SYSTEMS TESTED ON --

System 1
-Mac OS X 10.10.1
-JavaSE 1.6
-Java 8 Update 25

System 2
-Windows 10 v6.4.9879
-JavaSE 1.6
-Java 8 Update 25


-- RUN --

Do not touch the organization of packages. Simply run run "battlefield.mas2j" in Jason, the simulator will then run.
